# cloudflare-dyndns-rust

Reimplementation of [cloudflare-dyndns](https://github.com/colemickens/cloudflare-dyndns) in [Rust](https://www.rust-lang.org).

Builds with `rust 1.2.0 stable`.

## building

1. install `make` and `rust 1.2.0`.

2. clone this repo

3. `make`
